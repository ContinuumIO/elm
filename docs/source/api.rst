API
===

The API help information in this page is autogenerated from comments in the source.

The packages include:

 * :ref:`readers-api`, a collection of tools for reading HDF4, HDF5, NetCDF and GeoTiff files and directories of files
 * :ref:`pipeline-api`, the public interface for ensemble learning models (`elm`) and its `Pipeline` class for preprocessing transform steps before fitting in ensemble or in an evolutionary algorithm
 * ``elm.model_selection``, tools for scoring models, Pareto sorting models, and selecting the best ensemble members
 * :ref:`sample-util-api`, modules of helper functions to define classes / functions for ``elm.pipeline.steps`` (the possible steps in a ``Pipeline``)
